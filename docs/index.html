About This Repository

This repository contains two self-contained RISC-V bare-metal projects that run on QEMU virt. No OS, no external firmware dependencies for Project 1. Both projects are fully debuggable step-by-step in Eclipse CDT via GDB hardware debugging.

── QEMU virt boot chain ────────────────────────────────────
0x1000 MROM stub (5 instructions, hardcoded by QEMU)
↓ loads kernel_entry from 0x1018, jumps to it
0x80000000 Project 1: _start → main (M-mode, -bios none)
↓ or
0x80000000 Project 2: zsbl.S (M-mode ZSBL, -bios zsbl.elf)
↓ mret with MPP=S
0x80200000 Project 2: supervisor_main() (S-mode, SBI timer)

Prerequisites

You need these three tools installed before cloning either project.

RISC-V Toolchain
riscv64-unknown-elf-*

GCC cross-compiler, binutils assembler and linker for bare-metal RISC-V targets.

QEMU
qemu-system-riscv64

RISC-V system emulator. Version 6.x or later. Ships with most Linux distros.

GDB Multiarch
gdb-multiarch

Required for Eclipse CDT hardware debug. Can also be used standalone from terminal.

Eclipse CDT
eclipse-cdt (optional)

IDE for step-by-step debugging. Optional — make qemu-run works without it.

Install on Ubuntu / Debian

# QEMU and GDB sudo apt update sudo apt install qemu-system-misc gdb-multiarch make # RISC-V toolchain — two options: # Option A: distro package (easier, may be older) sudo apt install gcc-riscv64-unknown-elf binutils-riscv64-unknown-elf # Option B: SiFive prebuilt (recommended — matches this project) # Download from https://github.com/sifive/freedom-tools/releases # Extract and add bin/ to PATH in ~/.bashrc: export PATH="/path/to/riscv64-unknown-elf-toolchain/bin:$PATH"

Verify Installation

riscv64-unknown-elf-as --version # GNU assembler 2.x qemu-system-riscv64 --version # QEMU emulator version 6.x or later gdb-multiarch --version # GNU gdb

Quick Start

# Clone the repository git clone https://github.com/vwire/riscv-bare-metal-qemu.git cd riscv-bare-metal-qemu # ── Project 1: bare metal ────────────────────────────── cd bare_metal_qemu make qemu-run # builds and runs — loops silently (no UART output) # Ctrl-A X to quit QEMU # ── Project 2: ZSBL + Supervisor ────────────────────── cd ../zsbl_supervisor make qemu-run # builds and runs — prints full demo to terminal
